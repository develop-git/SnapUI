## Example

```
        let temp = UIView()
        temp.backgroundColor = .cyan
        temp.add(to: view)
            .margins()
```

## Installation

SnapUI is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby

pod 'SnapUI'
```
